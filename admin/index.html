⚙ API Setup

Step 1 — Create Google Sheet

  1. Go to sheets.google.com → New spreadsheet
  2. Name it: Literary Blog Database
  3. Create 3 tabs: Poems · Stories · Books
  4. Row 1 headers in each tab:
    id | title | date | content | excerpt | author | genre | status | image | featured

Step 2 — Deploy Apps Script

  1. In Google Sheet → Extensions → Apps Script
  2. Paste the full contents of admin/google-apps-script.js
  3. Replace YOUR_SPREADSHEET_ID with your Sheet ID (from the URL)
  4. Click Deploy → New deployment → Web app
  5. Execute as: Me · Access: Anyone → Deploy
  6. Authorize when prompted, copy the Web App URL

Step 3 — Connect & Test

Step 4 — Bake API URL into Site

After confirming the connection works, paste the same URL into _data/config.json in your project, then rebuild and redeploy. This makes content visible to all visitors, not just you.

{
  "apiUrl": "https://script.google.com/macros/s/YOUR_ID/exec",
  "siteName": "My Literary Space", …
}

Step 5 — Change Password

Password is stored as a secure SHA-256 hash in your browser.
